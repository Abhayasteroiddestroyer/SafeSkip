# 🚀 SafeSkip

**SafeSkip** is a smart attendance planner designed to help students manage their classes effectively. Calculate exactly how many holidays you can take without falling below your university's detention criteria.

## ✨ Features
* **Dark Mode UI:** Easy on the eyes for late-night planning.
* **Smart Calculation:** Tells you how many classes you can "SafeSkip" or how many more you need to attend to stay safe.
* **Customizable Criteria:** Set your target between 60% and 80%.

## 🛠️ Tech Stack
* HTML5
* CSS3 (Custom Dark Theme)
* JavaScript (Logic & Calculations)
